# Blood Donor Finder

## Community Impact Project

Blood Donor Finder is a community-driven web application designed to connect blood donors with recipients during emergencies. The platform helps users quickly locate compatible donors based on blood group and location.

## Problem Statement

Finding blood donors during emergencies can be time-consuming and challenging. This project aims to simplify the process by providing a centralized platform for donor registration and donor search.

## Features

* Donor Registration
* Blood Group Search
* Location-Based Donor Search
* Emergency Blood Request System
* Donor Contact Information
* User-Friendly Interface
